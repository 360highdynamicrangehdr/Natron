CheckerBoardOFX
===============

.. figure:: net.sf.openfx.CheckerBoardPlugin.png
   :alt: 

*This documentation is for version 1.0 of CheckerBoardOFX.*

Generate an image with a checkerboard. A frame range may be specified for operators that need it.
