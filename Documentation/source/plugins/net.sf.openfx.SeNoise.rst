SeNoise
=======

*This documentation is for version 1.0 of SeNoise.*

Generate noise.

Inputs
------

+----------+---------------+------------+
| Input    | Description   | Optional   |
+==========+===============+============+
| Source   | Source        | Yes        |
+----------+---------------+------------+
| Mask     | Mask          | Yes        |
+----------+---------------+------------+
