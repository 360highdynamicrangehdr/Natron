Natron is an open-source (MPLv2 license) video compositing software, similar in functionality to Adobe After Effects or Nuke by The Foundry. It is cross-platform (Linux, OS X, Microsoft Windows).

The project home page is http://natron.inria.fr The project source code repository is https://github.com/MrKepzie/Natron

Compositing is the process of blending real or virtual images together in order to produce an output image. It has a wide range of applications but mainly this is used in the visual effects production.

In order to composite images, you define image transformations, or operators, such as a box blur and you connect and blend the transformations together to produce a final image.

The chain of transformation applied to an image is called a compositing tree. An example of a compositing tree would be the following:

  • Read image A

  • Read Image B

  • Merge A over B with alpha blending

  • Apply a box blur on the result

In Natron we support OpenFX which defines a standard for writing visual effects/image processing plug-ins for a compositing software. This is really important as today over 500 plug-in exist and are widely used by the cinema industry. Those plug-ins can implement cutting edge computer vision/graphics algorithms and this is where Natron can become helpful for the research and education in computer vision/graphics:

500 lines of code are generally enough to implement an OpenFX plug-in with multi-threading support. If you're worried about performances, GPU acceleration is possible with 3rd party libraries.

Once a plug-in has been developed you can couple it with other plug-ins and watch the results. Also it will work on all the softwares that currently support OpenFX, namely:

  • Nuke by the Foundry

  • Sony Vegas

  • Assimilate SCRATCH

  • DaVinci resolve by Blackmagic Design

  • DustBuster

In order to be able to be ready for production of visual effects on movies Natron has a few other features:

  • Animation of visual effects over the time

  • Performances and ability to deal with very large images (tested on 27000x15000 images)

  • Dozens of image/video file formats supported.

  • Color management via OpenColorIO

  • Command-line support for render farm purpose

As of today we're working on a rotoscoping node to be able to create and edit masks in Natron. Please check out the full list of features and download Natron at: https://natron.inria.fr
