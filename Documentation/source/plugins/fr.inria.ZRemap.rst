ZRemap
======

.. figure:: fr.inria.ZRemap.png
   :alt: 

*This documentation is for version 1.0 of ZRemap.*

Remap Z-Depth pass according to a close limit value and a far limit

Inputs
------

+----------+---------------+------------+
| Input    | Description   | Optional   |
+==========+===============+============+
| Source   |               | No         |
+----------+---------------+------------+
