| Input    | Description   | Optional   |

--------

+-------------------+---------------+-----------+-----------------+---------------------------------------------------------------------------------------------------------+
| Amount            | amount        | Double    | 60              | Swirl amount.                                                                                           |
+-------------------+---------------+-----------+-----------------+---------------------------------------------------------------------------------------------------------+
